# Aplikasi Al-Quran 30 Juz 

Aplikasi Al-Quran 30 juz ini dibangunkan oleh  OPS WTG 1 untuk memudahkan umat Islam membaca dan mempelajari Al-Quran. 

Aplikasi ini menyediakan teks Al-Quran lengkap 30 juz, terjemahan, dan fitur-fitur lain yang bermanfaat.

Fitur-fitur Utama
1  Teks Al-Quran 30 juz lengkap dengan terjemahan Bahasa Indonesia.
2. Fitur pencarian surah dan ayat.
3. Fitur bookmark untuk menandai ayat-ayat penting.
4. Fitur audio murottal dari beberapa qari terkenal.
5. Tampilan antarmuka yang sederhana dan mudah digunakan.
